# Self-Hosted Music Library — Data Management Scripts

This project contains the scripts used to manage the data of a self-hosted music server: registering new songs into a SQLite database with their metadata, and tagging/classifying them through an interactive CLI, all synced with the server over SFTP.

It's part of a larger self-hosted music streaming project (Docker + Liquidsoap + nginx), and was also a way to put my knowledge of databases and SQLite3 into practice.
